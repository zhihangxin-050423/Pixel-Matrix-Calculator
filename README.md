# 像素矩阵计算器 · Pixel Matrix Calculator

> 将文字或图片转换为二维像素网格，自动计算并输出每个需要染色的方块坐标。

---

## 功能简介

这是一个运行于浏览器的单文件应用（纯 HTML + CSS + JS，无需安装任何依赖）。你可以输入任意文字或上传图片，程序会把它映射到一张自定义大小的像素网格上，并告诉你哪些格子需要染色、染什么颜色。

**典型使用场景**

- 在 Minecraft、像素画画布等网格类游戏中还原文字或图案
- 生成乐高拼图、十字绣、马赛克贴砖的坐标参考
- 教学演示像素化与颜色量化的原理

---

## 快速开始

无需安装，直接用浏览器打开即可：

```bash
# 克隆后双击文件，或：
open pixel_art_generator.html
```

## 主要功能

### 输入模式

| 模式 | 说明 |
|------|------|
| **文字** | 输入任意字符串，可自选颜色、字体、字重，程序自动计算最优字号填满网格 |
| **图片** | 上传 PNG / JPG / GIF / WEBP，自动按比例适配到目标网格尺寸 |

### 网格参数

- **宽度 / 高度**：独立设置，范围 4–200 格
- **颜色量化级数**：控制每个颜色通道的离散层级数（默认 6）。值越小，颜色种类越少，风格越趋向复古像素感
- **白色阈值**：亮度高于此值的格子视为空白，不参与输出（默认 240）
- **Floyd-Steinberg 抖动**：开启后用误差扩散算法重新分配量化误差，在有限色系下更好地还原渐变与细节

### 输出

- **可视预览**：实时渲染像素网格，格子足够大时自动显示网格线
- **统计面板**：总格数、染色格数、颜色种类、覆盖率
- **坐标列表**：逐行展示 `(x, y)` 坐标与对应颜色（HEX + RGB）
- **导出格式**：支持复制到剪贴板，或下载为 `JSON` / `CSV` / `TXT` 三种格式

---

## 坐标系说明

```
(0,0) ─────────────── (W-1, 0)
  │                       │
  │     像素网格区域        │
  │                       │
(0,H-1) ──────────── (W-1,H-1)
```

- 原点 `(0, 0)` 位于左上角
- `x` 轴向右递增，`y` 轴向下递增

---

## 算法流程

```
输入（文字 / 图片）
       │
       ▼
  高分辨率画布渲染
  （超采样 10–20×）
       │
       ▼
  双线性插值降采样
  → 目标网格尺寸
       │
       ▼
  颜色量化
  （简单量化 or Floyd-Steinberg 抖动）
       │
       ▼
  扫描非白色格子
  → 输出坐标列表
```

## 导出格式示例

**JSON**
```json
{
  "grid": { "width": 32, "height": 32 },
  "colored_cells": 186,
  "coordinates": [
    { "x": 4, "y": 7, "r": 230, "g": 57, "b": 70, "hex": "#E63946" }
  ]
}
```

**CSV**
```
x,y,r,g,b,hex
4,7,230,57,70,#E63946
```

**TXT**
```
(  4,   7)  ->  #E63946  rgb(230,57,70)
```
